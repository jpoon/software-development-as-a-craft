Programmer vs Developer

  • Software development is a craft.
  • Programming is only one aspect of software development.
  • Knowing how to program is not the same as being able to develop high quality software.


    It's like woodworking. You can START by growing a tree, then chopping it down and finishing it, sanding it, before you make a table. Or you can go to Ikea. More likely you'll try something in between. Modifying a WordPress theme is going to Ikea. Writing you own web framework is growing a tree first because you don't like the existing trees. You have to decide where on the spectrum you want to be, from being a custom furniture maker from the Woodright's Shop or someone who assembles prefabricated pieces made by someone else.

Software Development Process

  1. work closely with the customer to discover what is required
  2. design a software architecture that balances simplicity, scalability, performance, and reliability
  3. implement writing quality code
